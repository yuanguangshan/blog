越狱能力是一个人最根本的能力。<br/> <br/>  为什么大多数人一生苦苦挣扎却依然穷困潦倒？我来揭示一个真理，因为大多数人并没有越狱。<br/> <br/>越什么狱？穷狱。<br/> <br/>大多数人都被无比坚固的穷狱囚禁终身。<br/> <br/>  其实你这辈子已经够拼够卷，够忙够乱的了，但你发现越努力就越需要更多的努力，到最后甚至越努力越潦倒，反而当初啥都不干还好点，对吧？那么我要“恭喜”你，你已掉进了穷狱。<br/> <br/>  穷狱似乎是整个族群无意识构筑的一个机制，即：发展需要燃料，而在AI彻底替代人力之前，你就是不可替代的燃料。<br/> <br/>但你为何心甘情愿当燃料呢？这就是整件事最诡异且最巧妙的地方：  燃料想要摆脱燃料命运的努力，正是燃料燃烧的过程。<br/> <br/>  下面我来解读这句话。<br/> <br/>  你想要摆脱燃料命运，于是，你头悬梁锥刺骨苦读想要找个体面工作，有一点点闲钱就学着别人投炒买，甚至不顾所有人反对去创业。<br/> <br/>殊不知等待你的只是无尽加班和降薪，是套牢，是天台。<br/> <br/>而这时，等待已久的“救世主”们出现了，培训、网课、大师、成功学、财富跃迁课、自由职业指南如刀枪剑戟林立，进一步把你最后几根毛剐得干干净净。<br/> <br/>其实，大多数人老早聚被锁死在了穷狱之中，目之所及都是别人想让他看到的；念之所至都是别人想让他认为的；行之所及都是别人想让他执行的，并不知道自己的卖身契已经签到了孙子辈，还真以为自己在知识和财富的海洋里遨游。<br/> <br/>其实，把自己鸡到最高学历、熬夜加班累坏身体、削尖脑袋挤主流社会、高杠杆博弈、复鸡自己的孩子等等这些“摆脱燃料的命运的努力”，都只不过是穷狱的粗壮钢筋而已。<br/> <br/>  你跟随着数以亿计和你一样的燃煤，茫茫然做着这一切，在不断被注入的助燃剂的推动下，释放净自己最后一卡热能，最终，只落得一身疾病、满心悲怆、债台高筑、晚景凄凉。<br/> <br/>  这难道就是你想要的人生吗？  那你为何终身没机会跳出内燃机，跳出小我，跳出穷狱？换个角度来说，皆因你被普世价值和普世玩法深刻控制，你并没有机会活出一个独特性，一个自由，一个创造，一个舒展，一个真正的不同，而这些，本才是人的真正价值啊。<br/> <br/>  但真相远不止于此。<br/> <br/>  从根本处来说，穷狱是世间一切悲剧，一切痛苦，一切罪行，一切毁灭，一切low行的渊薮，是仇恨、嫉妒、贪婪、贫穷、破产、疾病、灾难的大本营。<br/> <br/>这些，你的老师当然不会教给你，你的老板更是害怕你知晓。<br/> <br/>甚至所有书籍都在告诉你如何快速成功，但不告诉你什么才是真正的成功。<br/> <br/>所以，你从小甚至没任何选择，被动适应一个谎言的世界。<br/> <br/>及长，网贷、消费主义、游戏、长短视频、各种对立、各种热点、各种焦虑更是蜂拥而至，将你五花大绑，囚禁于虚幻而焦渴的内燃闭环之中。<br/> <br/>  所以亲爱的兄弟姐妹们，越狱后你才能真正长大。<br/> <br/>  你们知道吗，万丈红尘竟没几个真正的大人。<br/> <br/>为什么？越狱太痛苦、太沉重、太艰辛、太孤独了，做个被穷狱保护着的孩子，似乎更“安全”呢。<br/> <br/>  但，大家都如此，便对吗？  如果大家纷纷喝醉了奔向那悬崖，你是否也要惶惶然茫茫然跟着去受苦受难呢？如果越狱意味着真正的清醒、独立和自由，那么这份清醒、独立和自由是否值得你拼命一搏呢？  为自己越狱，为众生醒来，这样的人生，值得一过。<br/> <br/>这才是真正的文明、真正的发展。<br/> <br/>毕竟这世间除了物质还有精神，除了竞争还有相爱，除了共性还有特异性，除了宏大叙事还有个体的微渺和真实，对吧？  人这一生，悲欢离合，浮浮沉沉，真的很短暂。<br/> <br/>如果回头一看，这一生终究还是做了一点原本不敢想的事，发展出了一些从未奢望过的能耐，或竟做了自己的好朋友，从集体无意识的互害中跳出来认认真真堂堂正正做了一回人，这也算越狱了，庶几无憾哉。<br/> <br/>  现在，我们需要一些越狱的方法。<br/> <br/>接下来我要说的，如果能唤醒你的无量智性，那就太好了。<br/> <br/>  首先，你得建个信仰。<br/> <br/>信仰，信仰，信仰，这玩意儿怎么说呢，似乎太高级，又太抽象，或者有点假有点酸腐？  事实还真不是这样，信仰反而是最糙、最有力量、最自然、最接地的玩意儿了。<br/> <br/>你要知道，信仰是越狱的第一推动力，信了什么而不是什么都不信，至少意味着这个世界不是毫无意义的一坨翔，对吧？而穷狱里的人大多是没有信仰的，世界对他们来说，只是一团混沌。<br/> <br/>那信仰什么呢？你不必信任何宗教，我来告诉你信什么——信你自己。<br/> <br/>这是仅属于你自己的宇宙，是的，就这样简单粗暴，这世界永远不可能超出你的感官世界——思考一下这句话——没人能绕过你的感受来证明这世界是个与你无关的客体。<br/> <br/>那么在这个独一无二的宇宙中，你才是你自己唯一的救赎，你才是你自己最大的恩人，你才是你自己最坚实的信仰，所以当你开始关注自己的呼吸，关注自己的真实感受，关注自己的真实想法，你自然会学着爱自己和体谅自己，照顾自己和帮助自己，于是，你会发现一个最大的真相：你自己才是一切事物运作的根本基石。<br/> <br/>  这真相是否已让你合不拢下巴？是否和你之前被操纵的那些想法截然相反？是否颠覆了封闭你于穷狱之中的不可撼动的普世强势观念？这就对了！  当你知晓了这个真相，你就不大可能继续欺骗自己了。<br/> <br/>你的心里会时常升起两句话：“去他的吧”！以及“关我屁事”！  走到这一步，你就基本拥有越狱的动力了。<br/> <br/>接下来你将目光如炬看透身边所有集体无意识。<br/> <br/>一个人的时候老老实实，一群人喝点猫尿就跌跌撞撞脏话连篇；一个人的时候可可爱爱，一群人工作就勾心斗角无比下作；一个人本来过得好好的，到了所谓“七夕”“中秋”“过年”，就被商业裹挟出内心翻涌，犯各种大病。<br/> <br/>  是的，人总是渴求集体能接纳、认可和标签自己，这样，自己的存在似乎就有了某种先验的合理性，有了某种不证自明的合法性、安全性、价值感。<br/> <br/>但果真如此吗？非也，当你身处某个集体，你就必然长出某种相应的集体无意识。<br/> <br/>别嘴硬也别抗争，这正是咱这具远古进化而来的身体的本能——群居让种子留存，合作让人类飞升。<br/> <br/>但古语“君子和而不同”似乎戳穿了这个真相，“和”是结构性的和，是小系统顺应大系统的动态和谐；而“不同”才是核心，正是这个核心让你身处任何集体之中都能清醒地意识到：穷狱就在那里。<br/> <br/>穷狱，它根植于那深深的、不可见的、先决性的集体无意识之中；而自己能在多大程度看清并跳出这份集体无意识，就能实现多大程度的解放和自由。<br/> <br/>  常见的集体无意识有：无条件攻击其他集体及其他信念；兽性本能的激发；对权威的盲信；对权威带来的没有证明过程的集体目标的盲从；焦虑和恐惧的放大；竞争思维、等级思维、物化思维、二元对立这四大悖谬的泛滥——这些锁你找开锁匠都打不开，因为你并不能真实找到它们，但它们却比你的门锁还真实地锁死了你一生的命运。<br/> <br/>  更可怕的是，这些集体无意正在且将要通过网络更深地控制住人类。<br/> <br/>  所以我不得不略带悲哀地说，这才是真正的大势。<br/> <br/>  我不得不说，在这样的“信息二次元世界”之中，人类的创造性、自由度、效能感、主人意识以及那大自然进化顶端的无比神圣的杰作——爱之柔软，将如不可见之花冠逐渐枯萎于每个人头顶，而手机里的二进制信息将取代这朵花冠，成为穷狱荷枪实弹的守卫。<br/> <br/>  所以从现在起，你得对自己摄入的信息有充分的筛选。<br/> <br/>  知否，当你点开手机你就已然失去了自由？钱塘江大潮一样的新闻、热点、街谈巷议、俗人之争、八卦是非、攀比、评测、焦虑、臆断、扭曲、放大、断章取义、暴力、投射、愚痴、怀疑、贪婪、怨恨、轻慢和麻木冲刷着你的伏隔核，伏隔核又释放出大量神经递质进行奖赏，于是，你迅速成了被信息控住G点的瘾君子，最终，你将失去基本的自控力、判断力和行动力。<br/> <br/>所以，要越狱你得先学会屏蔽。<br/> <br/>你既是世界之主人，你就得为进入你世界的信息负起责任，对吗？你知道吗，放下挣扎反而能轻松越狱，当你放下手机，累了就睡饿了就吃，现实的温暖和原生的智慧将不期而至。<br/> <br/>  行至此处，你已经有了生长的基础空间。<br/> <br/>那么接下来请你想到什么事就立即去做，不要拖延，不要给原始的婴儿大脑、集体无意识、内燃机陷阱以及信息病毒留下任何入侵的机会。<br/> <br/>你知道吗，这么多年困住你的并不是现实的无奈、社会的残酷和人心的叵测，而是你自己的拖延。<br/> <br/>  你永远无法计算有多少优雅而神圣的时光在你白日梦窗口忽忽而过，又有多少曼妙的机会和大好的青春在你对往事的无限反刍之中白白流逝。<br/> <br/>当你陷入拖延，越来越大的自我攻击将如飓风破坏掉内在世界的春和景明，你将渐渐失去面对真实、面对未知、面对创造、面对挑战的敏感性和真实兴趣，而只对抱怨来劲。<br/> <br/>你知道吗，穷狱里充塞着怨声和哀叹，穷狱里最不缺的就是推卸责任和自怨自艾啊。<br/> <br/>  小心你的完美主义倾向，其实不管什么事，你亲自先去做一下，出了问题再到处搜一下，实在不行去图书馆检索一下，你就能得到一个基础的认知，这恐怕比你在网络上、语言中、头脑里徜徉一年还有用。<br/> <br/>  这是个最好的时代，好就好在双刃剑的信息的另一面——真实的机会、海量的资源、无数的经验、高妙的智慧都在那里，只要舍得探索，也可以说俯拾皆是。<br/> <br/>  其实，你随便乱做也比不做好，拿起来就做，错了不正好得到了第一手经验？你的完美主义说到底还是一种自卑，你怕失败，怕出丑，怕真实世界的运作规律出乎你的预期，打乱了你的二进制呆板思维。<br/> <br/>  小心你的对抗，做事要么就全然专注去做，要么就不做，千万不要被你的工资定义了你的能力，也不要被人云亦云的行业前景囿住了你的愿景，把手头的不论什么事都当做你成长的练习和成就的阶梯。<br/> <br/>  向上攀爬不是为了主管，拼命学习也不是为了加薪，只是为了对得起自己这一去不复返的年轻，对吗？你知道吗，全然接纳是唯一的做事方式，如果说你以后能做点啥不一样的东西，那一定是你与世界和解后的一种自然生长，否则不管你怎么蹦跶都在反生命，绝不可能成功。<br/> <br/>  走到这里，你既有了独立的头脑，又有了现实成长，但要真正越狱，你可能还得再经历四重天的锤炼。<br/> <br/>  大家都知道炒股要赚钱是反人性的，其实炒股和越狱没啥区别，真正锁住我们的最终还是基因层面的规避风险和小富即安的趋向——这其实也是人类发展的根本规律——多看点历史我们就知道，任何朝代、任何商业、任何团体、任何学说都大抵走着“摧毁-创造-鼎盛-守成-衰败”的循环往复的过程。<br/> <br/>  我给你一个最实在的建议，作为个体的你，最好将自己这一生都维持于“摧毁-创造”的动态，永远不要故步自封，永远不要盛极而衰。<br/> <br/>  哪四重天？  第一重天是面对不确定性。<br/> <br/>把自己的脑袋从自我欺骗的沙子中拔出来，勇敢面对自己人性中的懦弱、懒散、恐惧和因循守旧，勇敢面对现实的危机和新流，得到自己的真实定位、真实角色、真实方向。<br/> <br/>  第二重天是接纳不确定性。<br/> <br/>将生活和工作中的一切未知甚至危机视作必然，安然以待。<br/> <br/>  第三重天是拥抱不确定性。<br/> <br/>将危机和未知转化为机会，与之合作而非对抗。<br/> <br/>  第四重天是创造不确定性。<br/> <br/>反其道而行。<br/> <br/>怀疑什么就探索什么，恐惧什么就迎战什么。<br/> <br/>生命力方向所在，虽千万人吾往矣，空乏其身，然后动心忍性，增益其所不能。<br/> <br/>简单说，就是主动去冒险。<br/> <br/>  在穷狱之中人们一定会忘记自己的历史，人们会非常天真地认为，这一次一定有所不同，这，真是神奇而荒诞的想法。<br/> <br/>  但这样的想法难道不是以非常真实的面目呈现在互联网上吗？大家品着猫嗑着娱乐圈，就像生活在幻梦中一样，顽固地认为自己的未来一定不同于悲催的现在，更不同于那昏糊的过去，顽固地忘记了自己还有多少贷款要还，顽固地忘记了人生的大幕将落、悲剧接踵、穷途已至。<br/> <br/>  此刻，一切都在被质疑，一切都在瓦解，一切都在建构，一切都在重组。<br/> <br/>一切都在汇入新的意义，新的价值。<br/> <br/>而新的增长点在哪里？新的归宿在哪里？新的希望、新的信仰、新的地平线究竟在哪里？  这一切，只有当你越狱进入真实的万类霜天之后，才能豁然洞悉，这既是你生命的方向，也是你生命的答案。<br/> <br/>  世界是个玩具，请你自己去玩。<br/> <br/>  而在你自己去找到这个方向、揭晓这个答案之前，我再最后给你三个建议：  1.吃好饭。<br/> <br/>2.睡好觉。<br/> <br/>3.放松踏实去生活。<br/> <br/>  没了。<br/> <br/>  看起来很简单，是吗？但这是越狱的先决条件。<br/> <br/>  而就这简简单单的三条，请问又有几个人能真正做到？  注意，本文说的冒险以及拥抱或创造不确定性等概念指的不是盲目换工作、投资或创业；更多指突破自己的认知茧房和僵化思维以及穷狱模式等，让自己生长、发展、突破和创造。<br/>
